# Crafting Path

::: info TL;DR
Normal → Magic → Rare. Once Rare, always Rare. Magic items hold 2 mods, Rare items hold 6.
:::

Every crafted item follows the same rarity progression. Understanding this path is essential because it determines which currencies you can use at each stage.

## Rarity Progression

```
Normal (0 mods)
  │
  ├─ Transmute ─→ Magic (1 mod)
  │                  │
  │                  ├─ Augment ─→ Magic (2 mods)
  │                                   │
  │                                   ├─ Regal ─→ Rare (3 mods)
  │                                                  │
  │                                                  └─ Exalt / Chaos / Essence / Annul ─→ Rare (3–6 mods)
```

### Normal → Magic
Use an **Orb of Transmutation** to add exactly 1 modifier. The item becomes Magic.

### Magic (1 mod) → Magic (2 mods)
Use an **Orb of Augmentation** to add a 2nd modifier. The Augment can add either a prefix or a suffix — it draws from whichever side doesn't have a mod yet.

### Magic → Rare
Use a **Regal Orb** to upgrade rarity and add 1 more modifier. A 2-mod Magic item becomes a 3-mod Rare.

### Once Rare, Always Rare
**You cannot downgrade an item's rarity.** A Rare item stays Rare forever. You can remove mods one at a time with Annulment, but even a 1-mod Rare is still Rare.

This is important because some currencies (Transmute, Augment) only work on lower rarities. Once you're Rare, your toolkit changes.

## Mod Slot Limits

| Rarity | Prefix Slots | Suffix Slots | Total |
|---|:---:|:---:|:---:|
| Normal | 0 | 0 | 0 |
| Magic | 1 | 1 | 2 max |
| Rare | 3 | 3 | 6 max |

A Magic item can have at most **1 prefix + 1 suffix = 2 mods.** A Rare item can have up to **3 prefixes + 3 suffixes = 6 mods.**

## The Standard Crafting Path

For a typical 6-mod craft, the path is:

1. **Transmute** — Target the hardest-to-hit mod on the primary side
2. **Augment** — Target the 2nd mod (or accept a miss and restart)
3. **Regal** — Upgrade to Rare, adding the 3rd mod
4. **Refine** — Exalt, Chaos, and Annulment to fill and fix remaining slots

The advisor optimizes each step by selecting the best [currency tier](/crafting/currency-basics) and evaluating [essence shortcuts](/crafting/essences).

## Refinement Options (Rare Item)

Once Rare, you have several tools:

| Currency | Effect |
|---|---|
| **Exalted Orb** | Add 1 mod to an open slot (cross-slot — draws from both prefix and suffix pools) |
| **Chaos Orb** | Remove 1 random mod + add 1 new random mod |
| **Annulment** | Remove 1 random mod |
| **Essence** (Perfect/Corrupted) | Remove 1 random mod + add a guaranteed specific mod |
| **Omens** | Modify the behavior of the next currency used |

The [Side-First Strategy](/crafting/side-first-strategy) explains how these tools are combined into a coherent plan.
